# PDF Guidelines

- PDF title should be in the format: "Release Notes - [Version]" and should be taken from the title in the input file.
- Background color should be white
- Text color should be black
- Links should be blue
- PDF size should be A4
- PDF should have a header with the title and the date
- PDF should have a footer with the page number
- PDF should have a margin of 1 inch on all sides
- PDF should have a logo in the header, just above the title and it should be provided by the user.
